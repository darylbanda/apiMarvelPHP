## apiMarvelPHP
# Proyecto de la próxima película de Marvel
Este es un proyecto donde muestro la siguiente película de Marvel usando una API pública.

## ¿Qué aprendí?
- Trabajé con cURL para obtener datos de una API.
- Aprendí a manejar fechas en PHP y mostrarlas de forma adecuada en el frontend.
- Estoy aprendiendo cómo organizar PHP, HTML y CSS para proyectos más grandes.

## Cosas que debo mejorar:
- Organizar mejor la estructura de archivos.
- Mejorar el diseño y responsividad del sitio.
- Agregar más funciones a la página, como la posibilidad de mostrar más películas.

## Cómo ejecutar el proyecto
- Clona este repositorio.
- Abre el archivo `index.php` en tu servidor local o utiliza XAMPP.
